  ## Meubel-house page

This is a furniture-shop website built as graduation project using React.

### Technologies stack
* React
* SASS
* axios
* styled-components
* React Hook Form
* Redux Toolkit, Redux Thunk/Persist
* eslint(linter)
* prettier(code formating)
